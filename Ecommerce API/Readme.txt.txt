Firstly wait for the project to load itself
successfully.

1. Go to terminal and write cd.. to come out of the
   project.
2. Create your own virtual Environment and activate it.
3. After activating the Environment go inside the
   project by cd Projectname.
4. Go to database.py and give the credentials in like
   Host Name, Password, database name, User Name.
5. Install all the required packages from the
   Requirements.txt.
6. Run the project by this command;
   uvicorn main:app --reload
7. Enter docs after localhost in searchbar like this
   Hostname/docs.
8. Lastly Ecommerce API view will be visible to you.
